Lecture 6: Substitution and the Geometry of 2-D Maps

Theme: Exploring how substitutions in integration require a "correction factor" determined by the geometry of the transformation.

Review of Substitution in Single and Multiple Variables

Example 1: Single Variable Substitution

Integrate the following expression:

\[\int_{0}^{\sqrt{\pi}/2} x \cos(x^2) dx\]
Step 1: Perform the substitution.
Let \(u = x^2\), which implies \(du = 2x dx\).
Step 2: Update the limits of integration.
When \(x = 0\), \(u = 0^2 = 0\).
When \(x = \sqrt{\pi}/2\), \(u = (\sqrt{\pi}/2)^2 = \pi/4\). (Note: The handwritten notes indicate \(\pi/2\) as a calculated limit for a slightly different boundary).
Step 3: Evaluate the integral.
\[\int_{0}^{\pi/2} x \cos(u) \frac{du}{2x} = \int_{0}^{\pi/2} \frac{1}{2} \cos(u) du\] \[= \left[ \frac{1}{2} \sin(u) \right]_{u=0}^{u=\pi/2} = \frac{1}{2}(1) - 0 = \frac{1}{2}\]

In this problem, we needed a correction factor of \(2x\) when replacing \(dx\) by \(du\).

Example 2: Double Integral over a Parallelogram

Integrate \(f(x,y) = xy\) over the parallelogram with vertices \((0,0), (1,1), (1,4),\) and \((2,5)\).

Visual Description: A parallelogram in the \(xy\)-plane. The bottom-left vertex is at the origin \((0,0)\). The rightward edge follows the line \(y=x\) up to \((1,1)\). From \((1,1)\), the shape extends up to \((2,5)\). The leftward edge connects \((0,0)\) to \((1,4)\), and the top edge connects \((1,4)\) to \((2,5)\).

First, we determine the equations of the boundary lines:

  • Line I: Connects \((0,0)\) to \((1,4)\). Equation: \(y = 4x\)
  • Line II: Connects \((1,4)\) to \((2,5)\). Slope is \(\frac{5-4}{2-1} = 1\). Point-slope form: \(y-4 = 1(x-1) \implies y = x+3\)
  • Line III: Connects \((1,1)\) to \((2,5)\). Equation: \(y = 4x - 3\)
  • Line IV: Connects \((0,0)\) to \((1,1)\). Equation: \(y = x\)

Using standard vertical slicing, the region must be split into two integrals based on the \(x\)-intervals:

For \(0 \le x \le 1\), the boundaries are \(x \le y \le 4x\).

For \(1 \le x \le 2\), the boundaries are \(4x-3 \le y \le x+3\).

The resulting integral is:

\[\int_{0}^{1} \int_{x}^{4x} xy \, dy \, dx + \int_{1}^{2} \int_{4x-3}^{x+3} xy \, dy \, dx\]

A more efficient approach to the above problem involves transforming the coordinates to simplify the region of integration. By observing the boundary equations, we can define new variables:

Alternative Approach:
The boundaries can be rewritten as:
  • \(y - 4x = 0\) and \(y - 4x = -3\)
  • \(y - x = 3\) and \(y - x = 0\)
Let \(u = y - 4x\) and \(v = y - x\). This maps the parallelogram to a simple rectangle in the \(uv\)-plane defined by \(-3 \le u \le 0\) and \(0 \le v \le 3\).
Key Concept: When we replace \(dx \, dy\) with \(du \, dv\), we may require a correction factor to account for how the transformation stretches or shrinks the area.

Formal Definitions

Definition: One-to-One (Injective)

Given a function \(f: A \to B\), we say it is one-to-one if no two distinct inputs in \(A\) give the same output in \(B\).

Mathematically: If \(f(p) = f(q)\), then \(p = q\).

Definition: Onto (Surjective)

Given a function \(f: A \to B\), we say it is onto if every element \(b \in B\) can be obtained as an output for some input \(a \in A\).

Examples of Mappings

Problem 1: \(f(x) = 3x + 5\) on \([0, 2]\)

Let \(A = [0, 2]\) and \(B = [5, 11]\). Is \(f\) one-to-one? Is \(f\) onto?

One-to-one check: Suppose \(f(p) = f(q)\).
\(3p + 5 = 3q + 5 \implies 3p = 3q \implies p = q\).
Thus, \(f\) is one-to-one.

Onto check: Given \(p \in [5, 11]\), find \(q\) such that \(f(q) = p\).
\(3q + 5 = p \implies 3q = p - 5 \implies q = \frac{p-5}{3}\).
Since \(q\) exists in \([0, 2]\) for every \(p\), \(f\) is onto.

Problem 2: Square Function on \([-1, 1]\)

Consider \(g(x) = x^2\) where \(g: [-1, 1] \to [-1, 1]\).

One-to-one check: \(g(1) = 1^2 = 1\) and \(g(-1) = (-1)^2 = 1\).
Since \(g(1) = g(-1)\) but \(1 \neq -1\), \(g\) is not one-to-one.

Onto check: There is no \(p \in [-1, 1]\) such that \(g(p) = -1\).
Thus, \(g\) is not onto.

Note: If we change the codomain of \(g(x) = x^2\) to be \([0, 1]\), the function becomes onto, though it remains not one-to-one.

Example 4: 2nd Order Mapping Analysis

Consider the function \(f: \mathbb{R}^2 \longrightarrow \mathbb{R}^2\) defined by:

\[f(x, y) = (x+y, x-y)\]

We want to determine if this function is one-to-one and onto.

Is \(f(x, y)\) One-to-One?

Suppose we have two inputs \((p, q)\) and \((r, s)\) such that \(f(p, q) = f(r, s)\). This implies:

\[(p+q, p-q) = (r+s, r-s)\]

This gives us a system of equations:

  1. \(p + q = r + s\)
  2. \(p - q = r - s\)

Adding the equations: \(2p = 2r \implies p = r\).
Subtracting the equations: \(2q = 2s \implies q = s\).

Since \((p, q) = (r, s)\), the function f is one-to-one.

Is \(f(x, y)\) Onto?

Given any point \((u, v)\) in the codomain, can we find \((x, y)\) such that \(f(x, y) = (u, v)\)?

We solve the system:

  • \(x + y = u\)
  • \(x - y = v\)

Solving for \(x\) and \(y\):
\(x = \frac{u + v}{2}\) and \(y = \frac{u - v}{2}\).

Since we can find an input for any output, the function f is onto.

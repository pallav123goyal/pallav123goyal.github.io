Lecture 4: Swapping the Order of Integration

In multivariable calculus, it is often useful to change the order of integration in a double integral to simplify the computation. This process requires re-evaluating the boundaries of the integration domain.

Example 1: Linear Boundaries

Express the following double integral by swapping the order of integration:

\[ \int_{0}^{8} \int_{y/2}^{4} xy \, dx \, dy \]
Visual Description of Domain: The domain of integration is a triangle in the first quadrant of the xy-plane. It is bounded by the y-axis from \(y=0\) to \(y=8\), the vertical line \(x=4\), and the line \(x=y/2\) (or \(y=2x\)). The region is shaded between the line \(y=2x\) and the vertical line \(x=4\).
Step-by-Step Solution:
  1. Identify Current Bounds: The original bounds are \(0 \le y \le 8\) and \(\frac{y}{2} \le x \le 4\).
  2. Determine New Bounds: To switch the order, we treat \(x\) as the independent variable. Looking at the domain, \(x\) ranges from \(0\) to \(4\).
  3. Find Functional Bounds: For a fixed \(x\), the value of \(y\) ranges from the x-axis (\(y=0\)) up to the line \(y=2x\).
  4. Rewrite Integral: The resulting integral is: \[ \int_{0}^{4} \int_{0}^{2x} xy \, dy \, dx \]
Example 2: Trigonometric Integration

Evaluate the following integral by swapping the order of integration:

\[ \int_{0}^{1} \int_{y}^{1} \sin(x^2) \, dx \, dy \]
Warning: The function \(\sin(x^2)\) cannot be integrated with respect to \(x\) using elementary methods. Swapping the order is mandatory here.
Visual Description: The region is a triangle bounded by \(y=0\), \(x=1\), and the diagonal line \(y=x\). The integration originally goes horizontally from \(x=y\) to \(x=1\).
Solution:

By switching the order, the bounds become \(0 \le x \le 1\) and \(0 \le y \le x\). The integral becomes:

\[ \int_{0}^{1} \int_{0}^{x} \sin(x^2) \, dy \, dx \]

Since \(\sin(x^2)\) is constant with respect to \(y\):

\[ \int_{0}^{1} [y \sin(x^2)]_{y=0}^{y=x} \, dx = \int_{0}^{1} x \sin(x^2) \, dx \]

Using u-substitution with \(u = x^2\) and \(du = 2x \, dx\):

\[ \frac{1}{2} \int_{0}^{1} \sin(u) \, du = \frac{1}{2} [-\cos(u)]_{0}^{1} = \frac{1}{2}(1 - \cos(1)) \]

Triple Integrals

Triple integrals extend the concept of integration to three dimensions. While single integrals represent area under a curve and double integrals represent volume under a surface, triple integrals are performed over a three-dimensional domain \(R\).

Motivation: We never really need to plot the four-dimensional function itself; we only need to visualize and plot the domain of integration in 3D space.
Example 3: Integration over a Rectangular Box

Integrate \(f(x,y,z) = xyz^2\) over the region \(R = [0,1] \times [0,2] \times [0,3]\).

Visual Description: The domain is a rectangular prism (box) aligned with the axes. It extends from 0 to 1 on the x-axis, 0 to 2 on the y-axis, and 0 to 3 on the z-axis.
Calculation: \[ \int_{0}^{1} \int_{0}^{3} \int_{0}^{2} xyz^2 \, dy \, dz \, dx \] \[ = \int_{0}^{1} \int_{0}^{3} \left[ \frac{xyz^2 y^2}{2} \right]_{y=0}^{y=2} \, dz \, dx \] \[ = \int_{0}^{1} \int_{0}^{3} 2xz^2 \, dz \, dx = \int_{0}^{1} [ \frac{2}{3}xz^3 ]_{0}^{3} \, dx \] \[ = \int_{0}^{1} 18x \, dx = [9x^2]_{0}^{1} = 9 \]
